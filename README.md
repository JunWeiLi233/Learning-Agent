# Why This?

Current AI agents are sick of designing frontend for the website. Though many skills give AI agents a great capability to design cool design, but they are all same templates. Coders and designers need something new, highly customized one. So, here's one skill helps AI agents to learn a special part of design from a website, it can be a simply button or entire design of the website.

---

# What does this mean?

This skill tells AI agents to first research over how a design in the website can be implement, then breaks it to a consistent running task. 

First, it obstains the research results to the mock trial, AI agent will rebuild design by their own. 
    If it fails, then go back and research another approach
    If it succeeds, compares the quality of original website design
        If the quality doesn't match, returns to the design again until it matches exactly design of original website

Then, as the design succeed, Ai agents will be told to write down the design language to a markdown file -> so it can be implement again and again.

The amazing part of current design is, it is not designing for just copying the original design, the skill will ask the ai agent to "learn it", to give different approaches for current design, and the AI agent rates new approaches and pick one that's best for current project.
